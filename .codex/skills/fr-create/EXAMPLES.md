# Create Examples

本文件给 `fr-create` 提供示例索引，不在 `SKILL.md` 展开长说明。

## Example Map

- `references/create-flow.md`
  - 何时看：目标路径、重名冲突、模板选择顺序还不清楚

- `../fr-cpt/examples/README.md`
  - 何时看：新建的是 `.cpt`，创建完就要进入 XML 编辑
  - 重点：CPT 样本入口和常见 XML 块

  - 何时看：新建的是 `.fvs`，创建完就要进入大屏结构编辑
  - 重点：FVS 结构入口和图表 cookbook

## Usage

- 先看 `create-flow.md`
- 按目标类型转到 `fr-cpt` 或 `fr-fvs` 的示例入口
- 不在这里展开具体 XML 细节
