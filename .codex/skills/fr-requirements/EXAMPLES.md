# Deprecated Requirements Examples

本文件给 `fr-requirements` 的旧引用提供迁移入口，不继续扩展旧流程。

## Example Map

- `../fr-workflow/EXAMPLES.md`
  - 何时看：历史记录只提到 `fr-requirements`，但当前任务需要重新路由
  - 重点：先把旧需求翻译到现行 workflow
- `../fr-init/EXAMPLES.md`
  - 何时看：旧需求本质上是缺配置、配置失效、要重建 `.codex/fr-config.json`
  - 重点：初始化字段和边界
- `../fr-cpt/examples/README.md`
  - 何时看：旧需求已经明确是 `.cpt` 编辑
  - 重点：CPT XML 示例入口
- `../fr-fvs/examples/README.md`
  - 何时看：旧需求已经明确是 `.fvs` 编辑
  - 重点：FVS 结构和图表示例入口

## Usage

- 先把旧的 `fr-requirements` 叙述翻译成当前路由
- 再转到目标 active skill 的示例入口
- 不要在这个 deprecated skill 下新增流程说明
