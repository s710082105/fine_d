# FVS Examples

本目录沉淀 `fr-fvs` 可直接复用的示例入口，不修改主 `SKILL.md`。

## Live Remote Samples

- `live-remote-patterns.md`
  - 2026-04-03 从 `http://192.168.0.99:8075/webroot/decision` 实例提取的真实 FVS 结构
  - 覆盖压缩包布局、`editor.tpl`、`store` JSON、图表资源、组件化 `.ec`、主题配置
- `chart-type-cookbook.md`
  - FVS 图表类型速查
  - 覆盖饼图、柱状图、折线图、条形图、面积图、地图、仪表盘、组合图的识别方式和样例入口

## Usage

- 先看 `Sample Map` 选样本
- 需要判断图表类型时，先看 `chart-type-cookbook.md`
- 再按 `Archive Anatomy` 定位实际要改的文件
- 只有在结构判断不出来时，才回到远端继续抓新样本
