# Upload Sync Examples

本文件给 `fr-upload-sync` 提供示例索引，不在 `SKILL.md` 展开长说明。

## Example Map

- `references/publish-rules.md`
  - 何时看：需要确认上传边界、远端校验、交接条件
  - 重点：只能传已保存本地文件、上传后必须校验、校验通过后才交给浏览器复核
- `../fr-browser-review/EXAMPLES.md`
  - 何时看：上传验证已经通过，下一步要做浏览器证据
  - 重点：固定复核流程和输出模板

## Usage

- 先按 `publish-rules.md` 确认上传和校验边界
- 校验通过后，再转到 `fr-browser-review`
- 如果远端校验还没完成，不要提前做浏览器复核
