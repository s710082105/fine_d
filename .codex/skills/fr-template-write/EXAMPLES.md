# Deprecated Template Write Examples

本文件给 `fr-template-write` 的旧引用提供迁移入口，不继续扩展旧流程。

## Example Map

- `../fr-create/EXAMPLES.md`
  - 何时看：旧需求本质是先新建模板或确认目标路径
  - 重点：create flow 和后续分流
- `../fr-cpt/examples/README.md`
  - 何时看：旧需求本质是 `.cpt` 编辑
  - 重点：CPT XML 示例入口
- `../fr-fvs/examples/README.md`
  - 何时看：旧需求本质是 `.fvs` 编辑
  - 重点：FVS 结构和图表示例入口

## Usage

- 先把旧的“template-write”语义拆成 create / cpt / fvs
- 再转到对应 active skill 的示例入口
- 不要继续沿用旧的混合式 workflow
