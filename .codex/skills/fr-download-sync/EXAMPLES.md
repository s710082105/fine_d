# Download Sync Examples

本文件给 `fr-download-sync` 提供示例索引，不在 `SKILL.md` 展开长说明。

## Example Map

- `references/sync-rules.md`
  - 何时看：本地是否过期、目标路径是否合法、是否必须先拉远端还不清楚
  - 重点：拉取前提、路径约束、最小证据
- `../fr-cpt/examples/README.md`
  - 何时看：拉回来的目标是 `.cpt`，接下来要判断该看哪类 XML 示例
  - 重点：CPT 样本入口
- `../fr-fvs/examples/README.md`
  - 何时看：拉回来的目标是 `.fvs`，接下来要判断结构或图表类型
  - 重点：FVS 样本入口

## Usage

- 先看 `sync-rules.md`
- 拉取完成后，按文件类型转到 `fr-cpt` 或 `fr-fvs` 示例入口
- 如果本地基线已经确定且不需要拉远端，不要在这里停留
