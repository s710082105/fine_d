# Browser Review Examples

本文件给 `fr-browser-review` 提供示例索引，不在 `SKILL.md` 展开长流程。

## Example Map

- `references/browser-review-playbook.md`
  - 何时看：需要按固定顺序做登录、打开预览、执行查询、截图取证
  - 重点：`.cpt/.fvs` 预览 URL 规则、登录顺序、不要重复走首页导航
- `references/review-template.md`
  - 何时看：需要整理最终浏览器复核输出
  - 重点：证据字段模板

## Usage

- 先看 `browser-review-playbook.md`
- 页面打开并完成检查后，用 `review-template.md` 整理最终证据
- 如果还没有上传验证证据，先回到 `fr-upload-sync`
