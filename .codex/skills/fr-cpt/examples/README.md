# CPT Examples

本目录沉淀 `fr-cpt` 可直接复用的示例入口，不修改主 `SKILL.md`。

## Live Remote Samples

- `live-remote-patterns.md`
  - 2026-04-03 从 `http://192.168.0.99:8075/webroot/decision` 实例提取的真实 CPT 配置模式
  - 覆盖参数面板、条件高亮、汇总公式、样式表、图表 XML、单元格超链接

## Local Supplements

- `live-remote-patterns.md` 末尾附带本仓 `reportlets/doc` 中的 JS / 参数 / 图表 / 超链接补充入口
- 这些补充样例用于弥补远端业务样本里没有显式 JavaScript 块的空白

## Usage

- 先看 `Sample Map`，按“业务列表 / 官方综合模板 / 空白骨架”选样本
- 需要 SQL、参数面板、条件高亮、公式、样式、超链接时，直接看 `live-remote-patterns.md`
- 只有远端样本没覆盖时，再回到文末 `Local Supplements` 查本仓 `reportlets/doc` 补充
