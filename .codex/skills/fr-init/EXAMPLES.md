# Init Examples

本文件给 `fr-init` 提供示例索引，不在 `SKILL.md` 展开长说明。

## Example Map

- `references/init-fields.md`
  - 何时看：需要确认字段含义、校验规则、哪些问题属于或不属于 init
  - 重点：required fields、derived fields、out-of-scope 边界
- `../fr-workflow/EXAMPLES.md`
  - 何时看：初始化通过后，不确定下一步该走哪个 skill
  - 重点：workflow 路由入口

## Usage

- 收集字段或解释字段时，先看 `init-fields.md`
- 初始化通过后，转到 `fr-workflow/EXAMPLES.md`
- 不在这里展开状态检查、数据源探测或同步细节
