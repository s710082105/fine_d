# Routing Matrix

- Missing or stale project config: `fr-init`
- Need environment, bridge, remote directory, or connection probe: `fr-status-check`
- Need connection list, SQL preview, field scan, or dataset XML: `fr-db`
- Need a new CPT/FVS target path and collision check: `fr-create`
- Need local CPT edits: `fr-cpt`
- Need local FVS edits: `fr-fvs`
- Need remote pull: `fr-download-sync`
- Need remote push and verify: `fr-upload-sync`
- Need browser evidence after sync: `fr-browser-review`
